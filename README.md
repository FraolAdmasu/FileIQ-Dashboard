## 👨‍💻 Developer
[![GitHub](https://img.shields.io/badge/GitHub-FraolAdmasu-181717?style=for-the-badge&logo=github)](https://github.com/Fraol+Admasu) <br>
Full-Stack Developer | Data & Analytics Focus <br>
A modern, intelligent file analytics web application built with **Streamlit**, designed to analyze, visualize, and extract insights from file systems and structured datasets.

The system follows a clean architecture separating UI, processing logic, and analytics engine to ensure scalability and maintainability.

## 📌 Overview

This dashboard transforms raw file structures and datasets into actionable insights through automated analysis, visualization, and reporting.

## ⚙️ Core Features

### 📂 File System Intelligence

| Feature | Description |
|--------|-------------|
| ZIP File Upload | Upload ZIP archives for automatic extraction, scanning, and cleanup |
| Local Folder Scan | Desktop-style folder selection for real-time file analysis |
| Secure Processing | Temporary extraction with automatic cleanup for safety |

### 📊 Intelligent Analytics Dashboard

| Component | Function |
|----------|----------|
| KPI Metrics | Total files, total size, unique file types |
| Pie Chart | File type distribution analysis |
| Line Chart | File creation timeline trends |
| Bar Chart | Largest files by storage consumption |
| Data Viewer | Advanced filtering (search, type, date range) |
| Export System | Download filtered datasets as CSV |

### 📈 Excel & Dataset Intelligence

| Capability | Description |
|------------|-------------|
| Multi-format Support | `.xlsx`, `.xls`, `.csv` |
| Auto Profiling | Rows, columns, missing values detection |
| Data Type Detection | Numerical, categorical, and date inference |
| Auto Visualization | Histograms and categorical breakdown charts |

### 🎨 UI / UX System

| Design Element | Approach |
|---------------|----------|
| Styling System | Custom CSS override of Streamlit defaults |
| Visual Identity | Inspired by Notion & Power BI |
| Typography | Inter font with clean spacing |
| Layout System | Responsive grid-based structure |
| Experience Goal | Minimal, modern, and data-centric UI |

## 🧠 Tech Stack

### 📊 Technology Breakdown

| Layer | Tools Used | Purpose |
|------|------------|--------|
| Core Engine | Python, Streamlit | Application logic and dashboard framework |
| Data Processing | Pandas, NumPy | Data manipulation and numerical analysis |
| Visualization | Plotly, Matplotlib | Interactive and static data visualization |
| File Handling | Excel (OpenPyXL support) | Reading and processing spreadsheet data |
| UI / Styling | HTML, CSS | Custom UI enhancement and styling |
